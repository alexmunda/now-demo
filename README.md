# now-demo
Clone the project
```
git clone https://github.com/alexmunda/now-demo.git
```

Install now
```
npm install now -g
```

Install Elm
```
npm install elm -g
```

## now-serve
Build the project
```
npm install

elm-package install

npm run build
```

Deploy with now-serve
```
cd dist
now-serve
```
